    Meet WebRocket!

    • Fast and scalable, distributed WebSockets broker
    • Well documented, powerfull and easy to use
    • Multiplatform, supporting many programming languages
    • Open source or hosted version
    • With commercial support

    How it works?

    • Browsers

      All browsers supporting WebSockets or Flash can connect to WebRocket, subscribe channels, listen for and broadcast events or trigger backend operations.

    • Clients

      Backend clients can deal with authentication or manage channels and broadcast events to them.

    • Workers

      Backend workers can connect to WebRocket and listen for the triggered operations.

    • Broker

      WebRocket ties everything up - broadcasts data from the backend clients or browsers to all others on the channel, passes trigger requests to the workers and provides full and flexible management API.
