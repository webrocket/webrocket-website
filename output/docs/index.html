    Installation

    If you don't have WebRocket installed yet, follow the instructions for your operating system:

    • Linux / Unix: Debian / Ubuntu, Fedora / RHEL, Arch Linux, Unix (from source)
    • Mac OS X: Homebrew, Macports, from source
    • On the cloud: Amazon EC2, Heroku Cedar Stack
